Tiff Split/Merge Utility

A simple utility for merging or splitting TIFF files

Merge multiple files into a single file

  • Click [Open] and select the files that you wish to merge into a single TIFF file.
  • Select the [merge to single file] option in "the Conversion Settings".
  • Enter an Output file name. By default, the program uses the file name "converted_YYYYMMDD.tif" and saves to the current working directory.
  • Click [Convert] to merge and save all selected images.

Split one file into multiple files

  • Click [Open] and select a single file that you wish to split to multiple TIFF files.
  • Select the [split to multi-files] option in the "Conversion Settings".
  • Click [Convert] to split the TIFF file into multiple images.
** The split images will output to the same directory as the source image and will add the suffix "_0.tiff", "_2.tiff", etc.

Arrange files

You may want to save the files in a different order. To change the row order, click on a row and then press [Page Up] to move the item up, [Page Down] to move the item down, and [Delete] to remove an item from the list.

Starting over

Click the [Clear List] button to remove all rows from the list and start again.
